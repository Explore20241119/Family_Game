# Family Game

A small family game hub with kid-friendly browser games for study, learning, and fun on iPad and web.

## About

This repository is a collection of simple browser games made for family use.
The goal is to create games that children can play, enjoy, and improve together with parents.

## Current Game

### Forest Study Trail

`Forest Study Trail` is a forest-themed browser game where the player:

- walks through forest paths
- collects study snacks and gold stars
- avoids forest goblins
- unlocks animal friends
